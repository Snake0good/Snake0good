Hi 👋 I'm Jacob
===========================

Full Stack Developer
--------------------

I started my coding journey 6 years ago with Python. Since then, I've earned multiple certificates from Coursera, Google, and IBM in Data Analytics and Data Science. I'm a designer at heart and love the visual aspect of seeing my creations run in the browser.
